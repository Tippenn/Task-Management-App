# Task Management App

A simple client-side task management application built to practice JavaScript fundamentals, DOM manipulation, and persistent state handling using localStorage.

This project is part of my learning journey from game development into full-stack web development.

---

## 🚀 Features

- Create, update, and delete tasks
- Mark tasks as completed
- Persistent data using browser localStorage
- Real-time UI updates without page reload
- Clean and minimal user interface

---

## 🛠️ Tech Stack

- HTML5 – Structure
- CSS3 – Styling and layout
- JavaScript (Vanilla) – Application logic
- localStorage API – Data persistence

---

## 🎯 Purpose of the Project

This project was built to strengthen my understanding of:

- JavaScript fundamentals and DOM manipulation
- State management in a frontend application
- Building interactive user interfaces without frameworks
- Data persistence using browser storage

---

## 🧠 What I Learned

- How to manage application state in vanilla JavaScript
- How to structure a small frontend project
- How to create responsive and interactive UI components
- How browser storage (localStorage) can simulate simple persistence
